# Installation and launch of the game "Fifteen"

### Project setup

```
npm install
```

### Compiles and hot-reloads for development
```
npm serve
```

### Compiles and minifies for production
```
npm build
```

### Start server for production
```
npm start
```

The app will be available on localhost:4200.
